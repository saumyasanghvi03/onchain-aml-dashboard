### üìã Installation & Regulatory Configuration

#### **1. Secure Repository Deployment**
```bash
git clone https://github.com/saumyasanghvi03/onchain-aml-dashboard.git
cd onchain-aml-dashboard
```

#### **2. Compliance Dependencies Installation**
```bash
pip install -r requirements.txt
# Additional regulatory compliance libraries
pip install aml-compliance-toolkit web3-regulatory-framework
```

#### **3. Regulatory Environment Configuration**
```bash
# Core AML Configuration
export AML_COMPLIANCE_MODE="STRICT"
export FATF_GUIDELINES_VERSION="2023"
export SANCTIONS_SCREENING_ENABLED="true"

# API Configuration for Compliance Services
export COINMARKETCAP_API_KEY="your_cmc_pro_api_key"
export CHAINALYSIS_API_KEY="your_chainalysis_api_key"
export WORLDCHECK_API_KEY="your_thomson_reuters_api_key"

# Regulatory Reporting Configuration
export FINCEN_BSA_REPORTING="enabled"
export EU_AMLD5_COMPLIANCE="enabled"
export REGULATORY_JURISDICTION="US,EU,UK"
```

#### **4. Launch Compliance Dashboard**
```bash
streamlit run app.py --server.port 8501 --server.enableCORS false
```

#### **5. Access Secure Dashboard**
Navigate to `https://localhost:8501` (SSL/TLS required for production deployment)
